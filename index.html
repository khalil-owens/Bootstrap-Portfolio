Khalil Owens Portfolio

Hello, welcome to my web development portfolio. I'm a high school student who's interested about building interactive and fun websites though HTML, CSS, JavaScript. Over the past several months I've worked on and built vairous projects, that gave me real web experiences from scratch.

I've created a *color picker* that changes background colors with the click of a button, built a fully functional *calculator* that performs real-time math, and coded a classic *Rock, Paper, Scissors* game using buttons and conditional logic. One of my favorite project was the *Memory Card Game*, where I learned how to use arrays, event listeners, and DOM manipulation to flip cards and match emojis. I also built a fun *Roll Dice* game and a *Customizer App* where users could type their info into boxes and see personalized messages appear on screen. These projects helped me understand how JavaScript makes websites come alive, whether it's handling button clicks, changing page elements, or keeping score in a game.
